# 生成AIリテラシー学習室

生成AIパスポート試験対策のための非公式・無料の学習用Webアプリです。2026年2月試験より適用のGUGA公式シラバスを軸にしています。外部ライブラリを使わない静的サイトで、`index.html`を開くだけでも動作します。

## 学習機能

- 14カテゴリ・134問のオリジナル4択問題
- 14カテゴリすべての章別教材（概要、4節以上の本文、重要ポイント、引っかけ、出典）
- 教材の読了管理、次の未読章の自動提案、カテゴリ別読了マーク
- 各問題の前に表示する要点解説
- 合格までの5段階学習ガイドと1日20〜30分の学習ルーティン
- 未回答、復習、保存、弱点モード
- 問題・用語・カテゴリ検索と用語集
- 今日の目標、学習範囲、正答率、連続学習日数
- 20問クイック模試と60問・60分の本番形式模試
- 模試の一括採点、結果履歴、誤答の復習登録
- JSON形式の進捗書き出し・復元
- スマホで問題を優先表示し、解説・結果・次の問題へ自動スクロール
- スマホ下部に固定される前へ・解き直し・次へ操作
- 回答後に自分の回答と正解、関連ポイント、引っかけ注意を表示
- 「まだ不安／理解した」の自己評価による復習日の予約
- 間違いは即時、理解済み問題は数日後に表示する予定日ベースの復習
- LocalStorageによる端末内保存

目標正答率80%は、このサイト独自の学習基準です。公式の合格基準として表示しているものではありません。

## 内容の検証

```powershell
npm.cmd test
npm.cmd run check:sources
```

`npm test`は問題数、ID重複、4択形式、解説の有無、正解位置の偏り、2026年の主要トピック、HTMLとJavaScriptの参照整合性を検査します。`check:sources`は`sources.json`に登録した一次資料の到達性を確認します。

## Web公開

GitHubリポジトリの`main`ブランチへ配置し、リポジトリ設定の Pages で「GitHub Actions」を選ぶと、`.github/workflows/validate-and-deploy.yml`が検証後に公開します。検証に失敗した内容は公開されません。

ワークフローは毎日21時（日本時間）に一次資料へのリンクを自動確認します。制度・シラバス変更を検出した後の問題改訂には、人間による内容確認が必要です。生成AIによる無検証の自動公開は行いません。

チームの役割、承認フロー、リリース条件は[PROJECT.md](PROJECT.md)で管理します。

## 出典と権利

問題は学習用に作成したオリジナル問題で、本試験問題や市販問題集の転載ではありません。主な根拠資料は`sources.json`で管理しています。法律、著作権、個人情報、生成AIサービスの仕様は変更される可能性があるため、試験前に公式・公的な一次資料も確認してください。
