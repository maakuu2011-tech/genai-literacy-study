      Study guide

      合格までの学習順序

      最初から暗記しようとせず、要点を読んで問題を解き、間違いだけを繰り返します。まず全範囲を一周することを優先してください。

      1. 基礎を固めるAIの基礎 → AIの歴史 → 機械学習 → ディープラーニング
      2. 生成AIの仕組みを理解する生成AI → 生成モデル → RAG・検索拡張生成 → AIエージェント
      3. 使い方を覚えるプロンプト → ビジネス活用。目的・条件・出力形式を整理する
      4. 得点源のリスク分野を仕上げるリスク・倫理 → 法律・個人情報 → セキュリティ → 情報リテラシー
      5. 弱点復習から本番形式へ未回答をゼロにし、弱点を解き直した後、60問模試で80%以上を2回目指す

      1日20〜30分の進め方

      新規10問 → 間違いの解説を読む → 復習5問。週末に20問模試、試験前は60問模試を行います。
